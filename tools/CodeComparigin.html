How to Use

Read shortcuts and instructions for side-by-side comparison.

1. Add Code: Paste or load code into both panels.

2. Compare: Click "Find Differences" to analyze lines.

3. Merge: Hover over lines in the diff panel to merge individual lines.

4. Bulk Merge: Click "Merge All" to apply all differences.

Keyboard Shortcuts
  • Ctrl + D: Find Differences
  • Ctrl + S: Save Active Panel
  • Ctrl + C: Copy Active Panel
